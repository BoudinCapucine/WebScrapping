Welcome to our french castles search engine ! 

First you need to download all the following librairies so the app can work: 
pip install streamlit requests folium streamlit-folium wikipedia-api geopy beautifulsoup4 textblob pandas
or
!pip install streamlit requests folium streamlit-folium wikipedia-api geopy beautifulsoup4 textblob pandas

Then you should open Anaconda Prompt, go to the app folder using cd commands and then run: 
streamlit run app.py 

Enjoy !
